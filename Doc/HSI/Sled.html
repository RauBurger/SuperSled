Initialization Notes

Serial:
The serial connector, J2, supports two functions.
1) Serial connection to the Telemega using UART0_RX on bit PTD6 and UART0_TX on bit PTD7.
2) Serial communication to a host, with a cable that also works with Telemega J21, using UART1_TX on bit PTE0 and UART1_RX on bit PTE1.

In both cases the unused UART port bits must be GPIO inputs.

Pressure sensor:
I2C0_SCL needs to be enabled on PTB0 and I2C0_SDA must be enabled on PTB1.

LEDs:
The red LED, D2, is near the SD card. PTA5 must be set to maximum drive strength. 0=LED on.
The green LED, D3, is near the servo connectors. PTA19 must be set to maximum drive strength. 0=LED on.

SPI:
The SPI bus supports several options:
1) Slave to the Telemega.
2) Master to local FLASH, U4.
3) SD card communications.
These options are exclusive of each other either through board build options or IO re-programming.

Push buttons:
Push button I/Os must have pull ups enabled.
